# Flutter Notes App

## Features
- Firebase Email Authentication
- User-specific notes
- Create, Edit, Delete notes
- Search notes
- Session persistence

## Tech Stack
- Flutter
- Firebase Auth
- Cloud Firestore
- Provider

## Setup
1. Clone repo
2. Add google-services.json
3. flutter pub get
4. flutter run

## Firestore Schema
Collection: notes
Fields:
- title
- content
- userId
- created_at
- updated_at
